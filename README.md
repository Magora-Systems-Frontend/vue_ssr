# Vue ssr template
Technologies
----------

- [Vue](https://vuejs.org//)
- [Vue Router](https://router.vuejs.org/)
- [Vuex](https://vuex.vuejs.org/)
- [Vue SSR](https://ssr.vuejs.org/)
- [Webpack 4.x](https://github.com/webpack/webpack)

Install dependencies
--------------------

>npm install

Development
--------------------
>npm run dev

Production
------------------

Build static files for client and server

>npm run build

Start server

>npm run start
